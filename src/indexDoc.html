API Assets
  • thumb_downWhat Does Trump Think

    With this API we created a function that calls the API with an Ajax call and returns a custom insult using the user submitted "Perp" in the style of a Donald Trump insult and attaches itself to user submitted entry.

  • mapGoogle Maps

    With this API we created a function that autocompletes location for the user in the "submit" modal and upon submission includes a map of where the offending dump took place.

  • file_upload Web API Javascript FileReader

    This API which is included in most modern browsers allows the application to read the file. While not a technical an API in the scope of this project, it is an API and we are listing it!

  • lightbulb_outline Future Ideas

    Dog API - With this API we can pull random pictures of dogs and also autocomplete a drop down list of breeds.

Assets and Libraries
  • gradehandlebars.js

    With handlebars.js we were able to make a template and render it to the submission card.

  • gradeMaterialize

    Materialize is a CSS framework based on Material design language and uses a responsive front-end framework that provides the user with a crisp and clean experience.

  • grade Google Fonts

    With Google Fonts we are able to use a variety of fonts which give our page a unique and different feel.

As the user scrolls through Dog Park Douchebag additional posts are appended dynamically to the bottom of the page as the user reaches the end of the content submissions.

We use Materialize's built in Modal to record the users submission to Dog Park Douchebag.

We believe the when the user chooses to Shame someone on our site it will be at the time of the crime, not days later. So to capture that momoment, we are using javascript to populate the modal with the current date and time. If the user is submitting at a different date and time, they can replace it.
Using the "What Does Trump Think" API we randomly generate an insult using the user input for "Douche Nickname" and insult them in the style of a Donald Trump tweet.
Using jQuery we created a custom one to five pooprating system. Once a user selects a pooprating, the rating is locked in and the user is unable to vote again.
Using Google Maps APIs Place Autocomplete we are able to provide a type-ahead search box that allows the user to type just about anything and get a valid address to choose from.
Using Google Maps APIs and the selection made in the submission form, we are able to generate a custom map with a pushpin showing the location of where the offending dump happened.
